Regles des jeux

Démineur

Le but du jeu est de découvrir toutes les cases libres sans faire exploser les mines.
Pour libérer une case, faire un clic gauche (clic normal).
Pour marquer une mine, faire un clic droit, qui fera apparaître un drapeau.
Le compteur en haut à gauche indique le nombre de mines qu'il reste à trouver.
Le chiffre qui s'affiche sur les cases cliquées indique le nombre de mines se trouvant à proximité sur une distance de une case.

Mots fléchées

Comme son cousin les mots croisés, le but principal est de remplir toutes les cases de la grille qui délimite le périmètre de jeu.
Le joueur selectionne une case avec un clique normal et remplis les cases avec son clavier.
Aprés avoir fait cela le joueur devra completer la grille,quand le joueur a trouver un mot les cases correspondant au mot ne peuvent pas être changer.
